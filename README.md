# chess

pieces -

  P = Pawn
  
  N = Knight
  
  B = Bishop
  
  R = Rook
  
  Q = Queen
  
  K = King
  
  
  
  
  
making moves -

  d2 d4  = piece at column d, row 2 moves to column d, row 4 (if it's a valid move). will handle captures.
  
  0-0    = castle kingside
  
  0-0-0  = castle queenside
